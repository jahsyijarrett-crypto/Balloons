# Balloons This program comes with an almost complete Balloon class, which provides the blueprint for an inflatable balloon object, and a BalloonRunner class to create Balloon objects.
Classes & Objects → The Balloon class is the blueprint, and the balloons you create are objects.
Instance Variables → Things like the balloon’s radius and color.
Constructors → Used to create balloon objects with starting values.
Methods → Functions inside the class, like inflate() and toString().
Object Modification → Changing object properties after creation (inflating or changing color).
